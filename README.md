# Fronty URL Focuser

Fronty URL Focuser is a Chrome extension for [Fronty](https://github.com/Burday/fronty) which allows it to automatically focus the URL bar.

# Installing
There are two ways to install this


# First Way:

Step 1 : Clone this repo with `git clone https://github.com/Burday/Fronty-URL-Focuser`

Step 2 : Navigate to chrome://extensions

Step 3 : Enable Developer mode

Step 4 : Press Load Unpacked and select the folder that contains manifest.json (the master folder)


# Second Way:

Step 1: Go to [the releases](https://github.com/Burday/Fronty-URL-Focuser/releases) and download the latest release. You will only need the .crx file

Step 2: Go to chrome://extensions

Step 3: Enable Developer mode

Step 4: Drag and drop the .crx file.
